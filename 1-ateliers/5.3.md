# Atelier 5.3 : gestion des erreurs avec *try/catch/finally* et *throw*

## Énoncé

1. Reprenez tous vos ateliers pour rajouter la gestion des exceptions à l'aide des blocs *try/catch/finally* sur toutes les parties de votre code susceptibles de déclencher une exception.
2. Lancez des exceptions à l'aide du mot-clé *throw* dans votre code lorsque la donnée attendue n'est pas conforme ou qu'il y a un scenario menant à une erreur.
