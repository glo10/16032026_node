# Atelier 5.2 : middlewares

## Enoncé

1. Reprenez l'atelier 5.1
2. Ajoutez des ***middlewares***, par exemple pour intercepter les requêtes avec un paramètre numérique, par exemple `/teams/1` et vérifiez le format de ce paramètre avant de faire le traitement dédié.

- Autrement dit, si le paramètre est numérique, vous pouvez laisser passer la requête au prochain *middleware* pour retourner une réponse avec succès.
- Dans le cas contraire, renvoyez un JSON avec un message d'erreur et le code HTTP adequate.
