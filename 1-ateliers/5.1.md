# Atelier 5.1 : API REST

## Énoncé

1. Créez un nouveau projet avec Express **sans** ***CLI*** (avec *npm*)
2. Créez une ***API REST*** pour la **Ligue des champions 2025-2026 phase finale** pour répondre aux besoins ci-dessous, vous pouvez vous appuyer sur les informations du site [***EuroSport***](https://www.eurosport.fr/football/ligue-des-champions/calendrier-resultats.shtml) sur la compétition. Votre application écoute sur le port ***5100***

**PS : il n'est pas demandé de restituer tous les clubs et matchs, un pétit échantillon suffira amplement** 


| Méthode HTTP | Endpoint   | Ressources             |
| -------- | -------------- | ---------------------- |
| `GET`    | `/teams`       | Liste des équipes (clubs) |
| `GET`    | `/teams/:id`   | Un seul club |
| `POST`   | `/teams` | Ajouter une équipe       |
| `GET`    | `/games` | Liste de tous les matchs |
| `POST`   | `/games` | Ajouter un match       |
| `PUT`    | `/games/:id` | Mettre à jour un match |

- Pour la route ***POST /teams*** : il faut stocker les utilisateurs dans un fichier JSON.
- Pour simplifier : à chaque ajout d'une équipe, récupérez le contenu acutel du fichier JSON, puis ajoutez la nouvelle équipe dans votre tableau enfin insérez en écrasant le contenu initial.
- Pour parser le JSON (transformer le contenu textuel de votre fichier en objet JS), utilisez la fonction ***JSON.parse(data)*** avec *data* le contenu du fichier JSON.


### Structure des données de votre API

```json
{
  "teams": [
    { "id": 1, "name": "Real Madrid", "country": "Spain" },
    { "id": 2, "name": "Manchester City", "country": "UK" },
    { "id": 3, "name": "Paris Saint-Germain", "country": "France" },
    { "id": 4, "name": "Chelsea", "country": "UK" }
  ],
  "games": [
    { "id": 1, "home": 1, "away": 2, "score": "3-0", "date": "2026-03-11" },
    { "id": 2, "home": 3, "away": 4, "score": "5-2", "date": "2025-03-11" }
  ]
}
```

Ici les propriétés *home* et *away* stockent les identifiants des équipes par exemple pour le match avec l'*id* 1, c'est le *Real Madrid* qui a joué à domicile avec une victoire 3-0 face à la *Manchester City* (équipe à l'extérieur).
