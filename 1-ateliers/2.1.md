# Atelier 2.1 : modules

## Énoncé

1. Initialisez un nouveau projet à l'aide de *`npm init`*
2. À l'aide de *npm* (Node Package Manager), ajoutez le paquet [ESLINT](https://eslint.org/) à l'aide de la commande `npm init @eslint/config@latest`, laissez-vous guider en répondant successivement aux questions posées et à la fin dites "yes" pour installer les dépendances correspondantes à vos réponses.
Voici les réponses à choisir durant la configuration
- Ok to proceed? (y) y réponse ***"y"***
- ? What do you want to lint? ... réponse ***"JavaScript"***

- How would you like to use *ESLINT* ? réponse ***"To check syntax and find problems"***
- What type of modules does your project use ? ***"JavaScript modules (import/export)"***
- Which framework does your project use ? ***"None of these"***
- Does your project use TypeScript ? ***"No"***
- Where does your project run ? ***"Node"***
- Would you like to install them now ? ***"yes"***
- Which package manager do you want to use ? ***"npm"***
3. Installez le module *Concurrently* suivant sur tous les environnements/
4. Installez les modules suivants uniquement en environnement de développement :
- *Vitest*
- *Cypress*
- *Typescript*
