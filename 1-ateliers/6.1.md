# Atelir 6.1 : *MongoDB et Mongoose*

## Installation MongoDB

### Depuis le Cloud

Pour créer une nouvelle base de données MongoDB Atlas, il faut : 
1. S’inscrire depuis [https://account.mongodb.com/account/register](https://account.mongodb.com/account/register)
2. Créez un cluster (machine virtuelle pour héberger vos futures bases de données), choisissez l’offre Free avec un stockage de 512Mo
3. Ajoutez votre IP pour restreindre la connexion uniquement depuis votre IP, si vous passez par un VPN, choisissez l’option « Allow Access from Anywhere »
4. Ajoutez le nom et le mot de passe de l’utilisateur
5. Choisissez le type de connexion « MongoDB for VS Code »
6. Copiez l’adresse de connexion qui sera affiché, elle vous servira pour vous connecter sur votre application et sur l’extension MongoDB de VS Code
***mongodb+srv://${id}:${password}@${cluster}/${name}?retryWrites=true&w=majority***

### Ou depuis Docker

1. Installez et lancez Docker
2. Copiez/collez les fichiers [docker-compose.yml](./ressources/6.1/docker-compose.yml) et [.env.example](./ressources/6.1/.env.example) à la racine de votre projet
3. Renommez ***.env.example*** en ***.env***
4. Modifiez uniquement les informations suivantes si nécessaire
- *MONGO_ROOT_USERNAME=champions_league*
- *MONGO_ROOT_PASSWORD=strong_password*
- *MONGO_DB_NAME=champions_league*
5. Lancez la commande suivante pour créer et démarrer vos containers
```bash
docker compose up --build -d
```

## Énoncé

1. Reprenez [l'atelier 5.1 sur la ligue des champions](./5.1.md).
2. Créez les schémas *Mongoose* pour les entités suivantes **sans la clé *id*** pour laisser à MongoDB de générer automatiquement la clé *_id* :
- *team* pour l'équipe
- *game* pour le match
- *player* pour le joueur
3. Implémentez les actions ***CRUD (create, read, update and delete)*** suivantes pour stocker les informations dans la base de données MongoDB : 
- L'ajout d'une nouvelle équipe dans la collection teams
- La modification d'une équipe
- La supprission d'une équipe
- La récupération d'une équipe à partir de son nom
