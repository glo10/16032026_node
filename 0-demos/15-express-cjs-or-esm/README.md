# Express

## Option 1 avec ESM (EcmaScript)

1. Partir d'un projet vierge ou déja existant avec le standard ESM
2. `npm i express`
3. Dans le package.json définier *"type":"module"* ou avoir les extensions *.mjs*

---

## Option 2 avec CommonJS

### Scénario 1 : projet vierge

1. Installez Express depuis le CLI
- Génère les fichiers avec le standard CommonJS
2. Codez en CommonJS

## Scénario 2 : projet vierge + travailler avec ESM

1. Installez Express depuis le CLI
- Génère les fichiers avec le standard CommonJS
2. Vous laissez les fichiers générés par Express en CommonJS
3. Sur les nouveaux fichiers, créez vos fichiers avec l'extension .mjs et codez avec le standard ESM.
